## Features

- Dinamic Background
- Change units for the temperature
- Loading screen
- API consume

## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceloGtz/Weather-App.git
```

Go to the project directory

```bash
  cd Weather-App
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
